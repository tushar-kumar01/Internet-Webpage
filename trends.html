WebAssembly and PWAs

WebAssembly improves performance, while Progressive Web Apps (PWAs) offer app-like experiences in browsers.

WebAssembly

  • Near-native performance in browsers
  • Compiled from C/C++/Rust
  • Ideal for graphics-intensive apps
  • Used in Figma, AutoCAD Web

Progressive Web Apps

  • Work offline
  • Installable on devices
  • Push notifications
  • Examples: Twitter Lite, Pinterest

Fun Fact

WebAssembly was first announced in 2015 and is now supported by all major browsers, enabling near-native performance for web applications.
