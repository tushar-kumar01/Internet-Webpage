The Birth of the Internet

ARPANET: The Foundation (1969)

Developed by the U.S. Defense Department's Advanced Research Projects Agency (ARPA), ARPANET was the first operational packet-switched network. It connected four university computers in 1969:

  • 1. UCLA - Leonard Kleinrock's lab
  • 2. Stanford Research Institute
  • 3. UC Santa Barbara
  • 4. University of Utah

Packet Switching Revolution

Unlike traditional phone systems that required dedicated circuits, ARPANET used packet switching - breaking data into small packets that could take different routes to their destination. This made the network:

  • More efficient - Shared bandwidth between multiple connections
  • More resilient - Automatic rerouting around failures
  • More scalable - Could grow without major redesign

TCP/IP: The Universal Language (1983)

The transition from NCP to TCP/IP in 1983 marked the Internet's true birth, creating a standardized way for different networks to communicate:

TCP Layer

  • Breaks data into numbered packets
  • Verifies complete delivery
  • Requests retransmission of lost packets
  • Reassembles data in correct order

IP Layer

  • Handles addressing (IPv4/IPv6)
  • Routes packets between networks
  • Fragments large packets when needed
  • Works with routing protocols like BGP

Why TCP/IP Won

TCP/IP succeeded because it was:

  • Open - Freely available specification
  • Flexible - Worked across different hardware
  • Scalable - Could handle network growth
  • Robust - Automatic error recovery

World Wide Web (1989)

Tim Berners-Lee at CERN invented the Web to help scientists share research. His 1989 proposal outlined:

  • HTML - Hypertext Markup Language
  • HTTP - Hypertext Transfer Protocol
  • URLs - Uniform Resource Locators

Web vs Internet

While often confused, they're distinct:

Internet

The global network infrastructure (since 1969)

  • Physical cables/routers
  • TCP/IP protocols
  • Underlying connection layer
World Wide Web

An application running on the Internet (since 1989)

  • Websites and pages
  • HTML/HTTP protocols
  • Content layer

Internet Evolution Timeline

1969

First ARPANET connection between UCLA and SRI. The first message: "LO" (attempted "LOGIN" that crashed after 2 characters).

1971

Ray Tomlinson invents email, choosing @ symbol. First network email program developed.

1983

TCP/IP becomes standard protocol. DNS (Domain Name System) introduced the next year.

1989

Tim Berners-Lee proposes the World Wide Web at CERN.

1991

First website goes live at CERN. Web becomes publicly available.
